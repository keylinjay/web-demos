CSS布局中包含一个重要的属性：position，它对应有如下四种属性值：

static

static 是默认值。任意 position: static; 的元素不会被定位。一个 static 元素表示它不会被“指定位置“，一个 position 属性被设置为”非static“的元素则会被定位。 

relative

relative 表现行为的和 static 一样，除非添加了一些额外的属性，例如，top，left，bottom，right。 

在一个相对定位（position属性的值为relative）的元素上设置 top 、 right 、 bottom 和 left 属性会使其偏离其正常位置。而页面其它元素不会调整位置来弥补它偏离后剩下的空间。

fixed

一个固定定位（position属性的值为fixed）元素会相对于window(浏览器视窗)来定位，这意味着即便页面滚动，它还是会停留在相同的位置。和 relative 一样， top 、 right 、 bottom 和 left 属性都可用。

小提示：移动浏览器对 fixed属性 的支持很差，不过这里有相应的解决方案

absolute

absolute 是最棘手的position值。 absolute 与 fixed 的表现类似，除了它不是相对于浏览器视窗而是相对于最近的可以被定位的父元素。如果绝对定位（position属性的值为absolute）的元素没有“可以被定位（position的值不是static)”的祖先元素，那么它是相对于文档的 body 元素，并且它会随着页面滚动而移动。

小提示：记住一个“可以被定位”的元素是指 position 值不是 static 的元素。
